# 黄宇航个人网站

## 📋 项目简介

这是一个现代化、响应式的个人网站，展示黄宇航作为全栈开发工程师的专业技能和项目经验。网站采用纯HTML、CSS、JavaScript开发，具有优秀的用户体验和视觉效果。

## 🎯 功能特色

### 🏠 首页 (index.html)
- **个人介绍**: 展示个人背景、技能专长和职业经历
- **技能展示**: 动态技能条，展示技术栈熟练度
- **项目展示**: 精选项目案例，包含跨境电商选品系统等
- **工作经历**: 时间线形式展示职业发展历程
- **联系方式**: 联系表单和社交媒体链接

### 📁 项目详情页 (projects.html)
- **项目概览**: 详细的项目介绍和技术栈
- **核心功能**: 系统功能模块展示
- **系统架构**: 技术架构图和数据流
- **项目统计**: 关键指标和技术亮点

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: CSS Grid, Flexbox, CSS动画
- **图标**: Font Awesome 6.0
- **字体**: Google Fonts (Inter)
- **响应式**: 移动端优先设计

## 📱 响应式设计

- **桌面端**: 1200px+ 完整布局
- **平板端**: 768px-1199px 适配布局
- **移动端**: <768px 移动优化布局

## 🎨 设计特色

### 视觉效果
- 渐变背景和毛玻璃效果
- 平滑滚动和动画过渡
- 悬停效果和交互反馈
