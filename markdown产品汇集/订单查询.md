## 📋 订单状态说明

### 订单状态流程
```
下单 → 支付 → 确认 → 生产 → 质检 → 发货 → 配送 → 安装 → 完成
```

### 详细状态说明

#### 1. 待支付
- **状态描述**：订单已创建，等待支付
- **操作建议**：及时完成支付
- **时间限制**：30分钟内完成支付

#### 2. 已支付
- **状态描述**：支付成功，订单确认
- **后续流程**：进入生产环节
- **预计时间**：1-2个工作日

#### 3. 生产中
- **状态描述**：产品正在制作中
- **生产周期**：15-30个工作日
- **进度查询**：可查看生产进度

#### 4. 质检中
- **状态描述**：产品制作完成，进行质量检验
- **检验内容**：材质、工艺、结构
- **检验时间**：2-3个工作日

#### 5. 已发货
- **状态描述**：产品已发出
- **物流信息**：可查看物流跟踪
- **配送时间**：3-7个工作日

#### 6. 配送中
- **状态描述**：产品正在配送途中
- **配送方式**：专业物流配送
- **配送范围**：全国主要城市

#### 7. 待安装
- **状态描述**：产品已到达，等待安装
- **安装预约**：联系客服预约安装
- **安装时间**：1-3个工作日

#### 8. 已完成
- **状态描述**：订单完成，产品已安装
- **质保开始**：质保期开始计算
- **售后服务**：享受质保服务

## 📱 查询步骤详解

### 官网查询步骤
1. **登录账户**
   - 访问 www.hongmu.com
   - 点击"登录"按钮
   - 输入用户名和密码

2. **进入订单页面**
   - 点击"我的账户"
   - 选择"我的订单"
   - 查看订单列表

3. **查看订单详情**
   - 点击订单号
   - 查看订单状态
   - 查看详细信息

### 手机APP查询步骤
1. **下载安装**
   - 应用商店搜索"红木家居"
   - 下载并安装APP
   - 注册账户

2. **登录查询**
   - 打开APP
   - 登录个人账户
   - 进入"订单中心"

3. **查看详情**
   - 选择要查询的订单
   - 查看订单状态
   - 查看物流信息
